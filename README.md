# Loan-Analysis-Power-BI-Dashboard
This project is a multi-page Power BI dashboard designed to analyse a financial institution’s loan portfolio from an executive perspective. It delivers actionable insights across risk exposure, customer segmentation, and financial performance.

**Project Overview**
The dashboard was built to support strategic decision-making by visualising key metrics, identifying high-risk segments, and uncovering revenue opportunities. It combines data storytelling, advanced DAX, and interactive visuals to create a polished, user-friendly experience.

**Dashboard Pages**

1. Executive Overview
High-level KPIs and loan trends, including total interest, loan count, default rate, and repayment distribution.

2. Risk Analysis
Credit risk segmentation by employment type, age, and credit score. Highlights default patterns and risk exposure.

3. Customer Demographics
Deep dive into borrower profiles based on age, income, education, employment type, and mortgage ownership.


**Key Features**
• Custom DAX measures for dynamic KPIs (e.g., Default Rate, Net Interest Revenue)
• Interactive slicers, drill-throughs, and tooltips
• Layout locking and visual consistency for executive readability
• Narrative insights and data-driven storytelling
